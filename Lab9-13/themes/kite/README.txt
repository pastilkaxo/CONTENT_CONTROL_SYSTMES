REQUIREMENTS
------------
Kite theme does not require anything beyond Drupal 8 core to work.

INSTALLATION
------------
1. Place "Kite" folder to the root /themes directory.
2. Login to the site and click on "Appearance" in the top Administration menu.
3. Click on "Install and set as default" next to Kite theme.

CONFIGURATION
-------------
Navigate to: Administration >> Appearance >> Settings >> Kite
